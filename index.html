INTRODUCTION

Building and hosting a simple web application is a great way to dive into web development and understand how websites work. In this project, I will create a basic web application using HTML, CSS, and simple JavaScript, focusing on both the frontend design and functionality. The goal is to build an intuitive and responsive website that can be easily accessed by users across different devices. Once the application is complete, I will host it on a free platform, making it publicly available with a custom subdomain. This project will provide a hands-on experience in both coding and web hosting, helping me gain a deeper understanding of web development
